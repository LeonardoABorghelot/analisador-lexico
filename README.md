### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/LeonardoABorghelot/analisador-lexico.git
cd analisador-lexico
```

### 2️⃣ Instalar as dependências

```
npm install
```

### 3️⃣ Rodar o servidor de desenvolvimento

```
npm run dev
```
